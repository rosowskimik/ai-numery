# SI - Rozpoznawanie Numerów

## Opis

Aplikacja, którą można nauczyć rozpoznawać odręcznie pisane cyfry od 0-9.

## Budowa z kodu źródłowego

1. Sklonuj repozytorium:

    * `git clone https://github.com/rosowskimik/ai-numery.git`

2. Zbuduj aplikację:

    * `cd ai-numery`
  
    * `cargo build --release`
